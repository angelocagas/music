# Angelo Cagas Music

A personal music portfolio website showcasing guitar cover videos — fingerstyle, electric, and solos.

> "Where words fail, music speaks." — Hans Christian Andersen

## Features

- Responsive navbar with smooth scroll navigation
- Hero section with banner image
- Feature highlights (Fingerstyle, Electric Guitar, Solos)
- YouTube video carousels for cover categories
- About section with social links
- Scroll reveal animations
- Back-to-top button
- Dynamic age and copyright year via JavaScript

## Tech Stack

- **HTML5** — Semantic markup and structure
- **CSS3** — Custom styles, animations, scroll reveal
- **JavaScript** — DOM manipulation, dynamic content
- **Bootstrap 4** — Responsive grid, navbar, carousel components
- **Font Awesome 6** — Icons
- **Google Fonts** — Montserrat, Ubuntu
